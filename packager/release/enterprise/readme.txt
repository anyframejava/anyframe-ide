Anyframe Java IDE(Enterprise Ed.), release 2.8.0 (2013.04)
------------------------------------------------
http://www.anyframe-ent.com

1. Anyframe Java IDE(Enterprise Ed.) 소개

Anyframe Java IDE(Enterprise Ed.)는 Anyframe Java Core 기반의 개발 환경을 구성할 수 있도록 Eclipse Plugin을 통해 프로젝트 구조 Scaffolding, 코드 생성 기능, Anyframe Plugin 설치 및 삭제 등을 지원하는 Community Edition과 함께, 
DAO Manager를 통해 Anyframe Enterprise 환경에서 DAO 소스 코드 개발을 UI 환경을 통해 쉽고 빠르게 개발할 수 있으며, 템플릿 기반의 자동생성을 통해 소스코드의 품질을 일정하게 유지 및 관리할 수 있다.

Anyframe Java IDE 관련 문서는 포탈 사이트의 문서 메뉴 내의 매뉴얼을 통해 확인할 수 있다.

* Anyframe Java IDE(Enterprise Ed.) 포탈 사이트 : http://www.anyframe-ent.com
* Anyframe Java IDE(Community Ed.) 포탈 사이트 : http://www.anyframejava.org

4. 라이센스 정책

Anyframe Java IDE(Community Ed.)는 라이센스 정책으로 Apache Licence, Version 2.0 (http://www.apache.org)을 채택한다. 단, Anyframe Java IDE(Community Ed.) 내에서 사용된 외부 오픈 소스의 경우 원 오픈 소스의 라이센스 정책을 따른다.
Anyframe Java IDE(Enterprise Ed.)는 라이센스 정책으로 Anyframe Java Enterprise Edition을 채택하며, 라이센스 정책은 license (Enterprise Edition).txt 파일을 참고하면 된다.
