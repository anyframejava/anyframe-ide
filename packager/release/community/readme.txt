Anyframe Java IDE, release 2.8.0 (2013.04)
------------------------------------------------
http://www.anyframejava.org

1. Anyframe Java IDE 소개

Anyframe Java IDE는 Anyframe Java Core 기반의 개발 환경을 구성할 수 있도록 Eclipse Plugin을 통해 프로젝트 구조 Scaffolding, 코드 생성 기능, Anyframe Plugin 설치 및 삭제 등을 지원한다.

Anyframe Java IDE 관련 문서는 포탈 사이트의 문서 메뉴 내의 매뉴얼을 통해 확인할 수 있다.

* Anyframe Java IDE 포탈 사이트 : http://www.anyframejava.org

2. Anyframe Java IDE Plugin 상세

1) Anyframe Java IDE - Eclipse Plugin
   . Common : org.anyframe.ide.common_2.8.0.jar
   . Code Generator : org.anyframe.ide.codegenerator_2.8.0.jar
   . Query Manager : org.anyframe.ide.querymanager_2.8.0.jar

2) Anyframe Java IDE - Ctip Integration
   . Ctip for Integration : anyframe-ide-ctip-integration-2.7.0.jar
   
3. 라이센스 정책

Anyframe Java IDE는 라이센스 정책으로 Apache Licence, Version 2.0 (http://www.apache.org)을 채택한다. 단, Anyframe Java IDE 내에서 사용된 외부 오픈 소스의 경우 원 오픈 소스의 라이센스 정책을 따른다.
