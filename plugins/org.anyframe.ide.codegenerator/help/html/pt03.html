III.How to use

Eclipse를 사용해서 Anyframe 기반의 프로젝트 생성, VO/Domain 클래스 및 CRUD 소스 코드 생성, DB 정보 설정 및 변경, Anyframe Plugin 설치/삭제, CTIP 연계 등의 작업을 할 수 있다. 프로젝트 및 클래스 소스 코드는 Eclipse Wizard를 통해서 생성되며 Anyframe Plugin 설치/삭제 및 CTIP 연계 등의 작업은 Eclipse View 화면을 통해 이루어진다.
