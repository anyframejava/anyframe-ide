5.시스템 사양

하드웨어와 소프트웨어 요구 사항은 다음 테이블에 작성되어 있다. 아래 작성된 하드웨어 사양 이하의 시스템에서도 Anyframe을 사용할 수 있으나, 빠른 성능을 위해서는 아래 사양을 권고한다. 또한 아래 작성된 Storage의 경우, Anyframe 전체 설치 파일에서부터 Eclipse, Tomcat, MiPlatform 등 모든 관련 파일 설치를 합하여 작성된 것으로 Anyframe 만으로는 1GB 이하의 용량을 차지하고 있다. 소프트웨어 요구 사항은 반드시 충족시켜줘야 한다.

아래 하드웨어 요구 사항은 Code Generator을 사용할 때 권고되는 사양으로, Code Generator를 사용하지 않고 Anyframe 기반의 어플리케이션 개발 시에는 아래 하드웨어 요구 사항을 따르지 않고, 제시된 사양에 비해 저사양에서 Anyframe을 사용할 수도 있다.
