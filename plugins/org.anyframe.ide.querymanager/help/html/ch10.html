10.Testing a SQL Statement

Query Test Editor는 Query ID를 추가하거나 편집할 경우 사용하는 에디터이다. 입력한 Query문에 대해서 테스트를 실행해 Result View에서 미리 결과를 확인할 수 있다.

  • Query Editor 구성

  • SQL Test 수행

10.1.Query Editor 구성

Query Test Editor는 다음과 같은 요소들로 구성되어 있다.

Query Test Editor의 주요 구성화면은 다음과 같다.

  • Query Information: Query ID, SQL문등 Query XML을 작성하기 위한 기본 자료 입력 기능 제공

  • Input Parameter: Query 테스트를 위한 데이터를 입력

  • Result Mapping: Select Qurey 결과값을 특정 Domain/VO 객체에 담을 경우 사용

  • Database Browser: 선택한 Profile Name의 Database Schema 정보를 트리 형태로 제공

10.2.SQL Test 수행

Query Test Editor에 정보를 입력 후 "Test Query" 버튼을 클릭하면 Query Results View 를 통해 결과값을 확인 할 수 있다 . Query Statement에 XML 주석처리구문(<!-- -->)이 존재하면 Test Query 수행 시 경고가 발생함으로 유의한다.
