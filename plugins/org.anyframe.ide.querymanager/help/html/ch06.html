6.Installation

Query Manager는 Eclipse Update Site와, Anyframe Portal에 압축 파일로 각각 배포하고 있다.

설치 전 확인

Query Manager는 내부적으로 Eclipse Data Tools Platform 플러그인을 사용하고 있으므로 해당 플러그인을 설치할 수 있는 환경을 구성해야 한다.

  • Eclipse Data Tools Platform 설치

    Eclipse Data Tools Platform은 Ecipse IDE for Java EE Developers Package에 기본 설치되어 있으며, 그렇지 않은 경우 각 Eclipse 버전에 따른 Update Site를 통해 설치할 수 있다.

  • Update Site에서 설치

  • 로컬 PC에서 설치

6.1.Update Site에서 설치

Eclipse의 Help > Install New Software 를 통해서 Anyframe Update Site인 http://dev.anyframejava.org/update 를 추가한다.

목록에서 (Anyframe Java IDE > Anyframe Java IDE Query Manager)를 선택하여 Query Manager를 설치한다.

6.2.로컬 PC에서 설치

압축파일로 배포되는 Query Manager는 다음과 같은 순서에 의하여 설치한다.

  1. Query Manager를 설치하기 위한 System Requirements 를 확인한다.

  2. Anyfrmae Java IDE의 최신 배포판 을 확인한다.

  3. Anyfrmae Java IDE의 최신 배포판("anyframe-ide-x.x.x.zip")을 사용자의 컴퓨터에 내려받는다.

  4. 내려받은 최신 배포판의 압축을 원하는 폴더에 푼다.

  5. 압축을 풀어서 나온 플러그인 패키지 파일("org.anyframe.ide.common_x.x.x.jar", "org.anyframe.ide.querymanager_x.x.x.jar")을 "[eclipse 설치 폴더]/dropins/anyframe/eclipse/plugins"나 "[eclipse 설치 폴더]/plugins"로 이동시킨다.

  6. Eclipse가 실행중이라면 다시 시작시킨다.

  7. "Help > About Eclipse > Installation Details , Plug-ins 탭" 를 통해 정상적으로 설치되었는 지 확인한다.

    ☞ About Eclipse 메뉴는 Windows용 Eclipse의 경우 Help 메뉴에서, Mac OS X용 Eclipse의 경우 Eclipse 메뉴에서 확인할 수 있다.

  8. Query Manager의 설치가 완료되었다.
