5.시스템 사양

하드웨어와 소프트웨어 요구 사항은 다음 테이블에 작성되어 있다. 아래 작성된 하드웨어 사양 이하의 시스템에서도 Query Manager를 사용할 수 있으나, 빠른 성능을 위해서는 아래 사양을 권고한다. 또한 아래 작성된 Storage의 경우, Anyframe Java Core 전체 설치 파일에서부터 Eclipse, Tomcat, MiPlatform 등 모든 관련 파일 설치를 합하여 작성된 것으로 Query Manager 만으로는 1GB 이하의 용량을 차지하고 있다. 소프트웨어 요구 사항은 반드시 충족시켜줘야 한다.

Storage 1GB 하드 디스크 공간

Query Manager 다국어 지원

Query Manager는 다국어 지원을 위해 NL(National Language) Resources를 구성하고 있으며 필요시 확장을 통해 다양한 언어 지원이 가능하다. 기본적으로 OS의 언어 구성을 따르며, 원하는 해당 언어로 Eclipse를 구동하고자 할 때는 다음의 설정 중 하나를 선택하여 참고 한다. 또한 매뉴얼의 이미지 및 메뉴 구성은 영어를 기준으로 작성되어 있다.

  • eclipse.ini 설정

    eclipse.ini 파일을 오픈하여 -Duser.language=en -Duser.country=US -Duser.variant=EN 을 추가 한다(영어기준).

  • 실행 시 옵션

    eclipse.exe -nl en_US 를 실행 파일에 추가하여 실행한다(영어기준).
