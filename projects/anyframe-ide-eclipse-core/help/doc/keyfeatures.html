Anyframe IDE User Guide    

Key Features

Anyframe IDE은 삼성SDS의 어플리케이션 프레임워크인 Anyframe을 기반으로 프로젝트를 개발할 때 프로젝트 생성 및 코드 생성 기능을 통해 개발 편의성을 제공해주는 개발 툴로 CLI(Command Line Interface) 혹은 Eclipse Plugin 툴을 통해 Scaffolding과 코드 생성 기능을 제공한다. Eclipse Help Contents를 통해서는 Eclipse Plugin 툴에 대한 사용법에 대해 주로 가이드하고 있다. Anyframe IDE은 다음과 같은 주요 특징을 가지고 있다.

Anyframe IDE Features

  • 타입별 프로젝트 및 빌드 스크립트 등을 생성한다.
  • 도메인 클래스를 중심으로 CRUD 기본 코드를 생성하고 테스트 코드와 테스트 데이터를 함께 생성한다.
  • 프로젝트개발 시 프로젝트 전체 구조 및 모듈, 소스 코드 생성 기능 및 프로젝트 단위 별 빌드 기능을 제공함으로써 신속한 개발 환경 구성을 가능하게 한다.
  • Anyframe을 사용하여 개발 시 직면하게 되는, 복잡한 설정 파일로 인한 잦은 오류 발생 상황을 줄일 수 있다.
  • Anyframe IDE에서 제공하는 코드 생성 기능을 이용하면, 개발 초기 공통 개발 template을 따로 구성하여 배포하고 각 개발자들은 배포된 개발 template 소스를 참조하여 수정해서 사용해야 하는 이러한 반복적인 작업이 불필요하다.
  • Best Practice에 해당하는 샘플 소스 코드를 자동 생성시켜줌으로써 Anyframe을 활용한 개발 시 예제 코드를 참조하기 용이하다.
  • 단위 test case 및 통합 test case를 자동 생성시켜줌으로써 개발 진행 시 테스트를 통한 코드 품질을 향상시킬 수 있다.
  • 코드 생성 시, 다음과 같은 공통기능이 제공된다.
    - Search (Primary Keys, Required Fields)
    - Paging (Paging Navigation Bar)
    - Exception / Message Handling
    - User Defined Template
