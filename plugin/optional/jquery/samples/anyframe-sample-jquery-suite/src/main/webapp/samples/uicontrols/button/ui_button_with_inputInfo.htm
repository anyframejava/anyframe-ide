Button 내부형태

  • submit : first name과 last name의 입력창에 값을 입력하여 submit 버튼을 누르면, 해당 값이 결과부분으로 텍스트로 보여진다.

Submit 선택자

		

		: submit 타입을 가지는 모든 요소들을 선택하여 적용할 수 있도록 만들어진 선택자이다.

		$(":submit") 과 같은 형식으로 선택자를 사용할 수 있으며, submit 선택자로 적용가능한 html tag는 다음의 3가지 경우이다.
		

		

1. <button></button>
2. <button type="submit"></button>
3. <input type="submit" /> 	
		

		
		
Submit 메소드

		

		: .submit()은 .on('submit', handler) 또는 .trigger('submit') 와 같은 동작을 구현할 수 있다.

		form에 이벤트 핸들러를 바인딩하여 사용할 수 있으며, 사용자가  form 전송을 할 때,  submit 이벤트가 일어나게 된다.

		.submit( [eventData], handler(eventObject) ) 이와 같은 매개변수를 사용하면, 이벤트 핸들러에 전달될 데이터 맵과 이벤트 핸들러로써 구현할 수도 있다.
