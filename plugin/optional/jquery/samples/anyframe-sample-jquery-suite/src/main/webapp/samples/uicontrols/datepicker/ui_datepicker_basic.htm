Datepicker 내부형태

: Datepicker 내부 형태를 변경시키는 요소들을 위의 체크박스로 나타난 옵션들을 통해 확인해 볼 수 있다.(다음의 옵션 확인)

  • other months : 한 달력에서 첫날과 마지막날에 이어지는 이전 달, 다음 달의 날짜가표현되어 나타난다.
  • selectable other months : 위 1번의 other months로 나타난 이전 달.다음 달의 날짜에 대해 선택가능하게 된다.
  • show week: 한 달력의 왼쪽에 몇 주째인지 숫자로 나타나게 한다.
  • button panel: 한 달력의 아랫부분에 버튼이 있는 바(Bar)가 나타나도록 한다.
  • dropdown month: 상단부분에서 정적으로 표기되어 있는 월 정보에 대해 콤보박스로 다른 달을 바로 선택해서 이동할 수 있도록 변경한다.
  • dropdown year: 상단부분에서 정적으로 표기되어 있는 년도 정보에 대해 콤보박스로 다른 달을 바로 선택해서 이동할 수 있도록 변경한다.
