[CSS : writing mode]
: css함수로 writing mode와 관련된 내용을 추가로 정의하여 가로/세로방향을 전환할 수 있다.
" -webkit-writing-mode " 이라는 속성명의 css 값은 다음의 5가지가 가능하다.
horizontal-tb | vertical-rl | vertical-lr | initial | inherit
세로형태로 텍스트가 보여지려면, -webkit-writing-mode를 vertical-lr 로 설정하면 된다.
