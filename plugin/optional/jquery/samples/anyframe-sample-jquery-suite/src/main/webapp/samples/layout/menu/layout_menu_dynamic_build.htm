동적구성

: 메뉴를 동적으로 구성하는 방식의 예와 하위 메뉴의 존재 여부를 알수 있는 아이콘을 변경하는 예를 보여준다.

Demo

Demo Description

  • 메뉴 재구성 Type1: 실제로 display 를 none 으로 구현 및 설정한 element tree 를 이용한 방법. 데이타는 이미 로딩되어 있음.
  • 메뉴 재구성 Type2: 동적으로 모든 메뉴 데이타를 추가 및 수정하는 방법. 해당 html 부분을 재 구성함.
  • 아이콘 변경: SubMenu 의 유무를 구분하는 아이콘에 대해서 3가지 형태로 구현한다. 아이콘이 없는 형태가 있으며, 계속 실행 시 아이콘이 변경되는 것을 볼 수 있다.

동적구성 방식

		    #58 ~ #59,#75 ~ #76
		    

			"$(element).html() or "$(element).html(newhtml)" 함수를 사용하여 해당 element 의 하위의 html 문자열을 Get/Set 한다. 

			그리고, 메뉴 변경  후에는 "$(element).menu('refresh');" refresh 메소드를 호출해야 한다.
		    

		    
		    
아이콘 변경

		    #83 ~ #84
		    

			icon GET: $(element).menu('option', 'icons') 

			icon SET: $(element).menu('option', 'icons', {submenu : icon정보} )

			본 예제에서는 Framework 이 들고 있는 아이콘 정보를 이용하였다. 아래 Framework Icon 링크 참조.
