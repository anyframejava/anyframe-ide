초기화

			

			ul, li, a 태그들로 구성한 후, container 가 되는 element 를 대상으로 "$(element).menu()" 와 같이 호출하면 된다.

			이 때 parameter 로 전달되는 값들은 초기화 시 사용된다.
			

			
			#45,#67
			

			
$("#hmenu").menu( { position: { my: "left top", at: "right+2 top+2" } } );
$("#vmenu").menu( { position: { my: "left top", at: "right+2 top+2" } } );			


			

			
			
			* 위의 형태는 오직 초기화 시 사용되는 parameter 형태임을 주의하자. 그 외, 각 attribute 의 get/set 은 아래와 같은 형태로 구현된다.

			

			
			#55,#57
			

			
var position = $(evt.target).menu('option', 'position');
$(evt.target).menu('option', 'position', position);			


			

			
			
동적 위치변경

			

			가로 메뉴의 경우 Level1 의 SubMenu 만을 Filtering 하여 기준 메뉴의 하단에 위치하도록 동적 구성 하였음 을 볼 수 있다.
