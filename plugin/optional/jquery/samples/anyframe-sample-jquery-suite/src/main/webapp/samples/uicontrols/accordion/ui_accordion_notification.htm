Code Description 

		

		
위치설정/변경

		#48 -  #52
		

		: position 메소드로 컴포넌트와 기준영역(혹은 다른 컴포넌트)과의 상대적인 위치를 정해 줄 수 있다. 
		position 내 3가지 옵션이 존재하며, 내용은 다음과 같다.

		
    
			
  • my위치시키고자 하는 해당 컴포넌트의 위치(해당컴포넌트의 어느위치가)
    • 
			
  • at기준영역 혹은 다른 컴포넌트의 위치(기준형태의 어느위치에 오도록)
    • 
			
  • of기준영역의 실제 타겟; 아이디값 혹은 클래스값 (기준이 될 타겟의)
    • 
		

		
		
헤더 아이콘 변경

		#36 -  #39
		

		: icons 옵션으로 다음과 같은 형태로 닫혀있을 때와 열려 있을 때의 아이콘 2개를 설정할 수 있다.

		
    
			
  • 닫혀있을 때"header" : 아이콘 클래스명
    • 
			
  • 열려있을 때"activeHeader" : 아이콘 클래스명
    • 
		

		
		
Accordion 헤더 CSS

		

		: accordion의 헤더에는 아이콘과 텍스트 부분이 분리되어 각각의 클래스가 존재하며 다음과 같은 각 클래스명을 통해 스타일 변경 가능하다.
		

		
    
			
  • 헤더 아이콘 부분: ui-accordion-header-icon
    • 
			
  • 헤더 텍스트 부분: ui-accordion-header
    •
