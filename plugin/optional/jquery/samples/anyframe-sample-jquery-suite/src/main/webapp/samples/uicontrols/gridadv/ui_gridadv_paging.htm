Demo Description

페이지 목록 보기 및 입력기능, Grid 에 보여질 최대 레코드 개수 선택 기능, 다음 페이지, 이전 페이지, 처음, 끝 페이지로 이동 등의 기능들을 구현하였다.

Code Description

			


            
초기화

            #56 ~ #59
            


rowNum: 3,
pager: ENV.ID_GRID_PAGER,
viewrecords: true,
rowList: [3, 5]


            

			

            Pager 구성을 위한 Option 들을 정의한다.

                
    
                    
  • pagerPager 가 표현될 Html Element 를 지정한다. 본 예제에서는 Div 의 id 를 지정하였다.
    • 
                    
  • rowNumGrid 에 나타낼 최대 레코드 개수, 즉 한 페이지 당 나타낼 최대 레코드 개수를 지정한다. (기본값 : 20)
    • 
                    
  • viewrecords조회된 전체 레코드 개수를 보여줄 지 여부를 지정한다. 본 예제에서 Grid 오른쪽 하단에 표현된다.
    • 
                    
  • rowListGrid 에 나타낼 최대 레코드 개수의 목록을 지정한다. (Array형태) 결국, rowNum 의 값을 동적으로 변경하게 된다.
