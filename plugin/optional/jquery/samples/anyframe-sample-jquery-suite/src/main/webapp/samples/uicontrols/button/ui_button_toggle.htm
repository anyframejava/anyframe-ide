Toggle 기능

: 토글로 조작할 수 있는 형태의 종류와 그 적용의 예를 보여준다.

  • toggle 기본 : 기본적으로 보여지거나 사라지는 것이 반복되는 토글기능 기본형태를 보여준다.
  • toggle 클래스 적용 : 배경색이 다르게 정의된 클래스들에 대해, 토글로 적용되는 형태를 보여준다.
  • toggle 효과 적용 : fade, slide, highlight 의 효과가 토글로 적용되는 예를 보여준다.

Code Description 

		

		
		
toggle 메소드

		#41 - #43  
		

		: 매칭되는 요소들을 보이거나 숨기게 하는 메소드로, 매개변수로 duration을 주면, 그 시간 만큼 느리게 혹은 빠르게 동작하게 된다.

		매개변수를 쓰지 않으면 애니메이션 없이 단순 css의 display  속성을 변화시킴으로써 숨기거나 보여지도록 만든다.
		

		
		
toggle 클래스적용

		#45 - #51  
		

		: 매칭되는 요소들에 하나이상의 클래스를 더하거나 지우는 역할을 한다. 

		매개변수로 클래스이름과 적용조건의 값을 주면, 그 조건의 결과가 true이면 클래스 이름을 적용하도록, false이면 적용하지 않도록 한다.
		

		
		
toggle 클래스 데이터 처리

		#16 - #18  
		#39 
		

		: 토글하기 위한 클래스들을 오브젝트의 배열로 처리하였다. 

		스타일 태그로 각 클래스별 스타일 정의 후, 클래스 명들을 배열 내  string으로 주면,   toggleClass("클래스명")과 같이 들어가는 부분에 배열로 그 데이터를 주게 된다.
