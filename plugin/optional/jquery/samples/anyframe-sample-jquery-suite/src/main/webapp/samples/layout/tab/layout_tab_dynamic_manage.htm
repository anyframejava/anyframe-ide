Layout - Tab

동적구성 및 Drag

: 동적으로 TAB 을 구성하고, TAB 버튼을 Drag 하여 위치를 변경할 수 있는 예를 보여준다.

Demo

Demo Description

탭(TAB)추가 버튼을 실행할 때 마다, 정해진 텍스트 값으로 신규 탭을 구성하며, 각 TAB 버튼에 있는 x 아이콘을 누르면 해당 탭을 삭제하게 된다.
TAB이동 체크박스가 선택되어 있을 때는, 생성된 탭 의 위치를 변경할 수 있다.

  • axis : 이 값은 X축 또는 Y축으로 이동을 제한시킬때 사용된다. 본 예에서는 X축 방향으로만 이동되도록 하였다.
    • 
				
  • stop : dragging 이 끝날때 발생되는 이벤트이다. 본 예제에서는 dragging 후, 다시 구성된 tab 정보를 반영하도록 'refresh' 메소드를 호출하였다.
