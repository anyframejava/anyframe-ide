Code Description 

		

		
append

		
  
		: append() 메소드로 표를 그리는 tr, td 와 같은 태그와 해당 데이터 내용을 추가시켜 줄 수 있다. 
				Html 소스를 추가시켜주므로, dialog에서 받은 값을 append 메소드로 추가시켜주는 것이다.
		

		
버튼 중앙정렬

		
  
		: 버튼바에 있는 버튼은 default로 오른쪽 정렬이 되어있다. 이를 중앙정렬 혹은 특정 위치 등으로 바꾸려면, 해당 버튼태그를 select하여 css를 수정하여야 한다.
 
		: dialog 아이디값을 이용하여 컴포넌트를 만들 때, 실제 전체 생성되는 컴포넌트 중 content 부분의 div 를 가리키게 된다. 

		: 따라서 버튼바 혹은	타이틀바를 수정하려면 dialog 아이디값의 prev() 혹은 next()를 선택하여야 한다.
 
		: 또한 하위에 생성되는 특정버튼 태그와 같은 경우, children()을 통해 접근할 수 있으며, 버튼의 중앙정렬을 위해서는  아래와 같이 css를 수정해주어야 한다.
