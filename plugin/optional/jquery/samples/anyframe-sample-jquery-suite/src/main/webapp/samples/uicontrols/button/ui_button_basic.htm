Button 기본기능

: Button 의 기본적인 기능으로, 4가지 기본적인 옵션을 사용한 컨트롤을 제공하는 예이다.

Demo Description

Button 내부형태

: 버튼 내부의 텍스트 사용여부, 레이블의 변경, 아이콘의 활용, 활성여부에 관해 옵션설정을 바꾸어보며 실제 시각적으로 바뀌는 것을 확인해 볼 수 있다.

  • 아이콘 : 버튼 내 아이콘과 텍스트가 함께 쓰이는 형태에 대해 4가지 타입을 확인해 볼 수 있다.
  • 라벨 : 버튼 내부의 라벨을 사용자 임의로 변경해 볼 수 있다.
  • 비활성화 버튼을 사용가능 혹은 불가능하도록 비활성화시킨다.

옵션: 초기화 설정

		
: 다음과 같이 button 컴포넌트를 처음에 생성할 때, 적용하고자 하는 옵션값을 설정할 수 있다.

		

			
$("#button").accordion({
	text : true,
	label : "변경할 텍스트",
	icons : { primary : "아이콘을 설정해 놓은 클래스명", secondary : "아이콘을 설정해 놓은 클래스명" },
	disabled : true
});	

		

		
(Default 값 : text : true  /  label: null/ icon : { primary: null, secondary: null } / disabled : false)

		
옵션: 재설정

		
 
		초기화된 후, 옵션을 재설정하여 사용가능하다. 본 예제에서도 초기생성 후, 옵션 재설정하여 사용하는 것에 대하여 구현해 두었다.
		

		

			
$("#button").button("option", "옵션이름", "옵션 값");
