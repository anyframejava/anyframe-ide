Button SET

: 라디오버튼이나 체크박스로 이루어진 버튼세트의 형태에 대해서 어떻게 컨트롤할 수 있는지를 보여준다.

  • Radio button : show/hide 로 구성되어 1개의 선택만이 가능하며, 선택에 따라 블럭이 보여지거나 사라진다.
  • Checkbox button : A/B/C/D 의 4개 버튼에 대해 체크된 알파벳만 각 결과화면에 text로 보여지게 된다.
  • Toolbar button : 여러 버튼들이 하나로 묶여진 툴바형태로 존재하며, 각 내부 버튼으로 결과화면의 텍스트들을 조작할 수 있다.
