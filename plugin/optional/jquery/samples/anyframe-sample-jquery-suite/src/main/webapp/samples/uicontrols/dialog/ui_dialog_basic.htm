Dialog의 구성요소

		
  
		: 다이얼로그는 크게  title, content, button bar 의 3개로 구성요소를 나누어 볼 수 있으며 텍스트 조작방법은 다음과 같다.
		

		
    
			
  • title bar 
			: div 태그 아래, 타이틀을 쓰는 span태그와 닫기버튼의 button 태그가 존재한다. 타이틀같은 경우, title  옵션으로 텍스트를 쉽게 정해 줄 수 있다.
    • 
			
  • content 
			:  div 태그 아래, p 태그로 써주게 되며, 본 예제에서는 동적으로 내용을 바꾸어 볼 수 있도록 이전 내용을 remove, 새로운 내용을 append 하도록 하였다.
    • 
			
  • button bar 
			: div 태그 아래, 여러개의 버튼을 묶어주는 div, 그리고 그 아래 button 태그가 존재하도록 틀이 정해져 있다. buttons 옵션을 통해 정해줄 수 있으며, 상세방법은 아래 옵션내용을 참조.
    • 
		

		
		
Dialog 옵션

		
  
		: 다음의 옵션들을 모두 true/false로 그 여부를 지정해 줄 수 있다.
		

		
    
			
  • Button bar 
			: buttons 옵션으로 여러개의 버튼과 기능을 구현할 수 있다. 즉, 버튼텍스트와 기능함수가 쌍으로 주어져야 하며, Array 와 Object 타입이 가능하다.
			
    • 
			
  • Buttons 
			:   [ {text : "button text 1", click : function(){...} }, {text : "button text 2", click : function(){...}  } ]
    • 
			
  • Modal 
			: modal 옵션을 true/false 로 설정하여 모달상태를 조작할 수 있다.
			
    • 
			
  • Resizable 
			: resizable 옵션을 true/false 로 설정하여 사이즈의 조절가능여부를 조작할 수 있다.
			
    • 
			
  • Draggable 
			: draggable 옵션을 true/false로 설정하여 다이얼로그 창을 드래그하여 움직이게 혹은 고정시키도록 조작할 수 있다.
			
    •
