Demo

Demo Description

다이얼로그

: Progressbar가 dialog안에서 나타나서 동작하게 된다.

  • dialog : start 버튼을 누르면 modal 상태로 다이얼로그가 나타나며, 그 안에서 progressbar가 나타나 진행상태를 보여주게 된다.
  • 진행상황 텍스트: 다이얼로그에서 진행되는 value를 % 단위의 텍스트로 보여지게 되며, 모든 진행이 완료되면 complete 텍스트가 나타나게 된다.

Code Description 

				

				
Dialog  내부요소 삭제/변경

				

				: 기본적으로 생성되는 dialog에 대하여 progressbar 만 나타나도록 내부적이 요소들이 삭제, 변환되어야 한다.
				

				
				
    
					
  • 타이틀 바
						: dailog의 타이틀 바는 id= "dialog"로 생성하는 것의 이전요소이다. 
						따라서 이를 삭제하기위해서는 다음과 같이  prev()로 요소선택을 한 후, remove() 메소드로 삭제해 주어야 한다.
						
    $(ENV.ID_DIALOG).prev().remove();
    
					
    • 
					
  • width 조정
					: dialog의 width 는 기본 300으로 값이 정해져 있으므로, 이를   auto로 값을 바꾸어야 한다.
