  • : 로딩 후 추가적으로, reserved target 명에 대한 데모를 실행해 볼 수 있다. (_blank, _self, _parent, _top)

Frame target names

			
    
				
  • "_blank"	: 새 창을 열어서 페이지를 로딩 한다.
    • 
				
  • "_self"	: 현재 frame 에 페이지를 로딩 한다.
    • 
				
  • "_parent"	: 현재 frame 이 포함된 frameset 의 부모 frameset 에 페이지를 로딩한다. 따라서 아래 예제에서의 _parent 는 최초 로딩된 페이지가 된다.
    • 
				
  • "_top"		: frame 이 나뉘기 전 original window 영역으로 페이지를 로딩한다. 동작방식은 "_blank" 와 유사하나, 현재 window 에 페이지가 로딩되는 차이가 있다.
