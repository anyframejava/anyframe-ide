Datepicker 날짜단위 및 형식 변경

: Datepicker 에 의해서 선택된 날짜가 표기되는 방식에 대해 날짜단위, 형식의 변경의 예를 보여준다.

Demo Description

Datepicker 표기방식

: datepicker로 선택한 날짜를 보여줄 때, 나타내는 표기방식을 각 단위,형식 변경을 통해 바꾸어 볼 수 있다.

  • 단위 변경 : 년, 월, 일 에 해당하는 날짜의 단위를 특정 문자구분단위 혹은 한자단위로 변경할 수 있다.
  • 형식 변경 : 날짜의 숫자값 자체를 형식을 다르게하여 변경할 수 있다.

날짜선택

		#60 ~ #69 
		
  
		: Datepicker에서 선택하는 순간 이루어질 동작들은 onSelect 옵션을 통해 다음과 같은 과정을 통해 함수를 작성할 수 있다.
		

		
    
			
  • date parameter 
			: date는 "월/일/년도" 와 같은 정보를 넘겨주며, string 값이다.
			따라서 이 값을 substr()과 같은 함수를 사용하여 각 년도, 월, 일의 값을 얻을 수 있다.
    • 
			
  • getDate
			: 자바스크립트의 일반적인 Date 오브젝트를 얻은 후, getMonth(), getFullYear(), getDate()로 각 년도, 월, 일의 단위숫자값을 얻을 수 있다.
    • 
			
  •  
			: 본 예제에서는 이와 같은 방식을 사용했으며, prop() 메소드로 각 입력창의 value값으로 이렇게 얻은 단위숫자값을 설정했다.
