Cell Editing

: Cell Editing 처리에 대한 예를 보여준다.

Demo

Demo Description

기본적인 골격은 inline editing 데모와 유사하며, 특히 cell editing 시 고려되어야 할 부분에 대하여 구현 되었다.
Cell Editing 적용을 위해, Add Row 실행 시 빈 데이타로 한 레코드가 추가되어 저장된다.

  • Edit Cell편집가능한 첫번째 레코드의 첫번째 컬럼의 셀 또는 마지막 편집했던 셀을 Editing 가능하게 변경한다. 마우스 클릭과 같은 효과.
  • Save Cell편집 중인 셀의 값을 저장하게 한다. Enter Key 를 실행한 것과 같은 효과.
  • Restore Cell편집 중인 셀의 값을, 원래 값으로 변경한다. ESC Key 를 실행한 것과 같은 효과.
