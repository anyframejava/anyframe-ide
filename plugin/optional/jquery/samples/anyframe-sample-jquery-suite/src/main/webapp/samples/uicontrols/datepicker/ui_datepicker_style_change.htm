Code Description 

		

		
스타일 지정 클래스

		
  
		: 각 스타일을 정의해 둔 클래스를 선택하여  css()로 변경하면 된다. 각 선택 클래스는 다음과 같다.
		

		
    
			
  • 헤더색 : (datepicker 아이디값) >div:first-child>div 
    • 
			
  • 라벨 텍스트색 : (datepicker 아이디값) >div>table>thead
    • 
			
  • 셀 내부색 (datepicker 아이디값) >div>table>tbody>tr>td>a
    • 
			
  • 셀 배경색(datepicker 아이디값) >div>table>tbody>tr>td>a
    •
