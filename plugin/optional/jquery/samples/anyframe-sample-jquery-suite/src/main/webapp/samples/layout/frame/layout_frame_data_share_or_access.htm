Data 공유 및 타 Frame Data 접근

: 여러 Frame 들 사이에서 Data 를 어떻게 공유 하거나 Access 할 수 있는 예를 보여준다.

Demo

Demo Description

초기 Frameset 로딩 페이지에 지정된 공유 객체에 데이터를 입력 또는 삭제 한 후, 변경 시 마다 해당 데이타를 list 로 표현한 샘플이다.

  • Left Frame 입력된 데이타를 공유 객체 에 추가한다. 공유 객체는 아래 "Code Source - Main" 부분을 참조하면 된다.
  • Right Frame 공유 객체에서 데이타를 삭제한다. (맨 뒤에서 부터..)
  • DataView Frame 입력/삭제 시 마다, list 데이타를 다시 그린다.

상위 Frame 접근 방법 (Child to Parent)

		    

		    "parent" 객체를 사용하면, 자신의 바로 상위 Frame 에 대한 접근이 가능하다. 아래 코드블록을 참조.		    
		    

		    

		    
parent.sharedObject.add( $("#tags").val() );
parent.sharedObject.del();
parent.sharedObject.getData();

		    

		    

		    위 각 코드는 Code Source 중 Left, Right, View 에서 발췌한 부분이며, 
 
		    모두 Code Source - Main 에 있는 Global Object 의 "sharedObject" 를 참조하고 있다.
		    

		    
		    
하위 Frame 접근 방법 (Parent to Child)

		    

		    frame 엘리먼트 정의 시, "name" 으로 지정된 값을 이용하여 접근이 가능하다. 아래 코드블록 참조.
		    

		    

		    
targetDataView.updateListView();


		    

		    
		    

		    본 예에서는 "targetDataView" 만을 하위로 접근하고 있으며, Code Source - View 의 코드 중 updateListView 함수를 호출하고 있다.
