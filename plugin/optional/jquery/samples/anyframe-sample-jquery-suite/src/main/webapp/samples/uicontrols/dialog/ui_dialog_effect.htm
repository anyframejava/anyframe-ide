Dialog 애니메이션 효과

: dialog 가 열리고 닫힐 때 나타날 수 있는 애니메이션 효과들을 적용해 본다. 또한, dialog를 drag할 때 적용될 수 있는 효과들의 예를 보여준다.

다이얼로그의 열림/닫힘 효과

: dialg가 열릴 때, 닫힐 때 나타날 수 있는 애니메이션 효과들과 drag로 다이얼로그를 움직이는 동안 나타날 효과를 조작해 본다.

  • 열림/닫힘 효과 : show/ blind/bounce/clip/drop/explode/fade/fold/hightlight/puff/pulsate/scale/shake/size/slide 가 적용가능하다.
  • 드래그 효과 : 드래그할 때, 투명도, 되돌아가기, 테두리의 효과를 주어 조작해 볼 수 있다.
