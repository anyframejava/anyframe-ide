Demo Description

Grid Inline 편집은 Row 단위로 Grid 에 직접적으로 입력 및 수정하는 것을 말한다.
위 데모는 추가, 수정, 저장 및 취소의 기능을 모두 버튼에 연결하였으므로, 저장과 취소 또한 해당 기능을 호출하여야 한다.

편집가능 여부 설정

			#51 ~ #58
			

			colModel 정의 시, Inline 편집이 가능하게 하려면, "editable" property 를 "true" 로 설정해 두어야 한다. 			

			"id" 컬럼의 경우는, rowid 와 맵핑된 키 정보이므로, "editable" 값을 "false" 로 설정해 두었고, 편집 또는 추가시 해당 컬럼이 Read-Only 상태인 것을 볼 수 있다. 
			

			
			
    
				
  • Add: "addRow" 메소드 호출. #75
    • 
				
  • Edit: "editRow" 메소드 호출. #80
    • 
				
  • Save: "saveRow" 메소드 호출. #85
    • 
				
  • Restore: "restoreRow" 메소드 호출. #96
    • 
			

			
			
메소드 호출규약

			

			Grid 의 각 메소드 호출 규약은 아래와 같은 두가지 형태를 갖고 있으며, 어떤 방식을 사용하여도 무방하다.
			

			


Type1. this._grid.jqGrid('addRow', addparams);
Type2. this._grid.jqGrid.addRow(rowid, addparams);

			 "id" 키 값의 유/무를 판별하여, 데이타를 추가 또는 수정한다.
