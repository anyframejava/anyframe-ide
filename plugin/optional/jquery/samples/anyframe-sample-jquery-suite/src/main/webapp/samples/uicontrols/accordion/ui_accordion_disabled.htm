Code Description 

		

		
비활성화 클래스 추가

		

		: 비활성화시키고자 하는 헤더의 인덱스를  eq()함수로 찾은 후, 해당 헤더에 addClass로 "ui-state-disabled" 라는 클래스를 추가시키게 된다.

		만약 비활성화 시키고자 하는 섹션이 이미 활성화되어 header와 panel이 모두 보여지고 있는 상태이면, 
		해당 header 바로 다음에 붙어 있는 panel도  비활성화 클래스를 적용시키도록 한다.(next()함수로 다음요소선택)
