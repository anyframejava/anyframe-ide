Accordion 기본기능

: accordion의 기본적인 기능으로, 두가지 옵션을 사용한 컨트롤을 제공하는 예이다.

Demo Description

Accordion 형태

: 각 섹션은 bar 처럼 생긴 헤더(header)부분이 존재하며, 클릭과 같은 이벤트로 하위콘텐트(content)부분이 된다.
이벤트의 종류나 상세 애니메이션 동작 등은 조절가능하며, 아래 형태 옵션 동작으로 세부형태는 바뀔 수 있다.

형태 옵션 동작

: collapsible, heightStyle 옵션조절을 통해 기본형태의 세부적인 부분을 바꿀 수 있다.

  • 모두닫힘 여부 : 적어도 하나의 콘텐트가 펼쳐져 있어야 하는지, 모든 콘텐트가 접혀져(닫아져) 있을 수 있는지 설정한다.
  • 이벤트 변경 : 아코디언이 작동하는 인터랙션의 이벤트로써, 마우스 클릭, 오버 등으로 변경 가능한다.
  • 현재창 제어 : 아코디언 내에서의 이벤트를 통하지 않고 외부의 다른 UI요소를 통해 열리고 닫힘을 제어할 수 있다.(본 예제에서는 버튼으로 제어)
  • heightStyle : 각 섹션의 콘텐트의 높이를 조작한다.
    • auto : 가장 많은 콘텐츠를 가지고 있는 섹션의 높이에 기초하여 그 최대높이값을 기준으로 모든 섹션에 적용하여 보여준다.
    • fill : accordion의 부모높이에 기초하여 가능한 높이로 보여준다.
    • content : 각 섹션의 콘텐츠의 길이에 맞추어 높이가 변하며, 이를 적용하여 보여준다.

Code Description 

		

		
Accordion 구조

		

		: accordion은 타이틀이 쓰여지는 header 부분과 그 아래 내용물이 들어있는 panel이 있다. 

		즉, (header + panel) 이 한 세트로써 동작하며, 이것이 여러개 나열되며 아코디언이 구성되는 것이다.
		

		
		
생성가능한 태그

		

		: 아코디언을 생성하기 위해 임의의 마크업 모두가 가능하다. 단지, 각 panel 은 header 바로 다음에 존재해야 한다.
		header 옵션에 header로 동작할 태그를 선택자로 넣어주어야 한다.
		
		
		
		
옵션: 초기화 설정

		
: 다음과 같이 accordion 컴포넌트를 처음에 생성할 때, 적용하고자 하는 옵션값을 설정할 수 있다.

		

			
$("#accordion").accordion({
	collapsible : true,
	heightStyle : "content"
});

		

		
(옵션 값을 따로 설정하지 않고 컴포넌트를 생성하면 default값으로,	collapsible은 false로, heightStyle은 "auto"로 설정된다.)

		
옵션: 재설정

		
 
		초기화된 후, 옵션을 재설정하여 사용가능하다. 본 예제 참조 : #29,#33 

		다음은 일반적인 옵션설정방식이다.
		

		

			
$("#accordion").accordion("option", "collapsible", true);
