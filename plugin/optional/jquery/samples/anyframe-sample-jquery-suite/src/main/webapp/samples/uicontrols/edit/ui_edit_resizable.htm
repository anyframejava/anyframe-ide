Resizable

		#28 - #35 
		

		: 하나의 위젯형태로 이루어진 컴포넌트처럼 $( ".selector" ).resizable() 과 같은 방식으로 어떤 요소의 사이즈를 마우스를 이용하여 변경해 볼 수 있다. 

		또한 변경가능한 옵션관련 주요사항은 다음과 같다.
		

		
    
			
  • 
				방향설정
				 handles 옵션: 사이즈의 조절을 할 방향을 설정한다. 가능한 값은  { n, e, s, w, ne, se, sw, nw } 이며, north, south, east, west 의 약자로 쓰였다.
			
    • 
			
  • 
				최대/최소너비 및 높이
				 minWidth, minHeight, maxWidth, maxHeight 옵션 : 사이즈 변경가능한 최소 또는 최대 너비/높이값의 조건을 숫자로 지정해 줄 수 있다.
			
    • 
			
  • 
				조절시 가이드라인
				 helper 옵션 : 사이즈조절을 위해 마우스로 드래그할 때, 아웃라인이 보여지도록 한다. 아웃라인의 스타일을 지정해 놓은 클래스명을 지정해 주면 된다.
			
    • 
			
  • 
				  본 예제에서는 .resizable-helper 클래스를 정의하여 적용시키도록 하였다.#15 - #17
			
  • 
				사이즈조절 동작
				 resize 이벤트 (resize(event, ui)): 사이즈를 조절하는 동안 여러가지 ui 요소들을 조작할 수 있는 이벤트함수이다. 
			
    • 
			
  • 
				 resizable이 가지고 있는 element, helper, size, position 등을 조작 가능
    
			
    • 
		

	
		
Resizable 위젯 구조

		

		: $(selector).resizable() 과 같이 사이즈조절가능한 형태로 구현하면, 브라우저에서는 실제 자동으로 해당 태그를 div 태그로 한번 더 감싸지는 형태로 형성된다.
		
		: 상위에싸여진 div 태그에 적용되는 클래스는  .ui-wrapper 이며, 여러가지 css 속성을 가지고 있다. 

		: 본 예제에서는 사이즈 조절표시를 input창 또는 textarea창 내부에 두기위하여 감싸진 상위 클래스의 padding-bottom을 0으로 재변경해 두었다.
		#37

		(default로 적용되는  padding-bottom 값은 16px)
