Demo Description

상단 TAB, 하단 TAB, Contents 영역인 Panel 부분이 접히는 유형을 구현하였다.
추가적으로 마우스 클릭이 아닌, MouseOver 시 에 도 같은 동작을 하는 예를 구현하였다.

Code Description

		    

		    
		    
초기화

		    

		    ul, li, a 태그들로 TAB 버튼 영역을 구현하고, Contents 영역인 패널부분은 Div 태그로 내용을 구현한다.

			container 가 되는 element 를 대상으로 "$(element).tabs()" 와 같이 호출하면 된다.

			이 때 parameter 로 전달되는 값들은 초기화 시 사용된다.
		    

		    
		    
상/하 위치

		    #117 ~ #150
		    

		    상단 TAB 과 하단 TAB의 모습은 해당 영역을 구성하는 ul 이하의 트리가 panel 영역 (div 영역) 의 앞/뒤 위치에 따라 결정된다.
		    

		    
			
탭 접기

		    #47 ~ #68
		    

		    display style 값, aria-expanded 및 aria-hidden attribute 값을 조정하여 구현되었다.
		    
		    
		    
		    
이벤트 변경

		    #80 ~ #87
		    

		    $(element).tabs("option", "event", "mouseover") 와 같이 호출하면 된다. "mouseover" 값 대신  "click" 을 입력하면 기본값으로 설정된다.
