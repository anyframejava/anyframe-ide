Code Description

		    

		    
		    
위치 변경

		    #51 ~ #78
		    

			Radio 버튼에 맞게 미리 지정된 position 값을 Menu 에 적용한다.
		    

		    
		    

		    position 객체 설명 (position : {my:"left top", at:"right top"})
		    

		    
    
		    	
  • 
		    		my: 현재 위치를 변경하는 대상에 대한  Align 기준 값. 즉, 네 모서리 중 하나. 본 예에서는 submenu 자체를 의미한다.
		    	
    • 
		    	
  • 
		    		at: 기준 대상에 대한 Align 기준 값. 본 예에서는 submenu 를 갖는 메뉴 아이템을 의미한다.
		    	
    • 	
		    

		    
	    
		    즉, 각 두 대상 Object 가 만나는 모서리(Align 면)을 고려하여 값을 지정하면 된다.
