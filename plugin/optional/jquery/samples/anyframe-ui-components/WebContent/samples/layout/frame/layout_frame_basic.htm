Frame 기본기능

: Frame 이 갖고 있는 기본적인 속성을 이용하여, 화면을 나누어 작업하는 예를 보여준다.

Demo

Demo Description

Frame Layout 을 구성하는 HTML 문서는 일반적인 HTML 문서의 구성과 다르게, BODY 태그 위치에 FRAMESET 및 FRAME 태그들로 구성된다.

  • Frame1, Frame2: "cols" attribute 를 이용하여, 가로를 30%, 70% 비율로 나누어 Frame 을 구성한다.
  • Frame3 - 6: marginwidth, marginheight attribute 를 적용한 예를 보여준다. IE 에서는 한 가지 값만 지정시, 기본값 적용 결과가 상이하게 나타나는 증상이 있다.
  • Frame7 - 9: noresize attribute 를 적용한 예를 보여주며, noresize attribute 의 경우에는 해당 attribute 의 존재 여부로 resize 여부를 결정한다.
  • : 기본값은 resizable 이다. Frame7 과 Frame8 사이는 수직 경계선은 noresize 가 적용되었고,
  • : Frame8 과 Frame9 사이의 수평 경계선은 resize 가 적용된 것을 확인 할 수 있다.
  • Frame10 - 12: frameborder attribute 의 값을 0 으로 설정하여, Frame 들 사이의 경계를 그려지지 않도록 한 예를 보여준다.

FRAMESET ELEMENT - 주요 attributes

			
    
				
  • - rows: 수평으로 배치될 프레임의 구성을 정의한다.
    
					: 각 값은 comma 로 분리하여 작성하며, 픽셀, 퍼센티지, 상대길이 값 등을 조합하여 작성된다. (기본값은 100%)
				
    • 
				
  • - cols: 수직으로 배치될 프레임의 구성을 정의한다. 적용되는 값은 rows 의 것과 같음.
				
    • 
			

			
			
FRAME ELEMENT - 주요 attributes

			
    
				
  • - name: target 프레임을 지정할 목적으로, 프레임의 이름을 정의 (타 프레임안의 데이타에 접근 시 사용됨)
    • 
				
  • - src: 해당 프레임에 로드 될 문서 위치 정보
    • 
				
  • - frameborder: 프레임 보더의 사용여부 결정. 0|1. (기본값은 1)
    • 
				
  • : IE 경우, 프레임들 사이의 공간을 명시하는 framespacing 속성을 지정해야 한다. (기본값은 2)
    • 
				
  • - marginwidth: 픽셀값. 프레임 가로 마진
    • 
				
  • - marginheight: 픽셀값. 프레임 세로 마진
    • 
				
  • - noresize: resize 여부를 결정. attribute 존재여부로 값을 결정함. (기본값은 resizable)
    • 
				
  • - scrolling: 스크롤 바 사용여부. yes|no|auto. (기본값은 auto)
    •
