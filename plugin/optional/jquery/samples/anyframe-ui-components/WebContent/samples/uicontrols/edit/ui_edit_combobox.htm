Demo Description

[Purpose]
: 입력창과 버튼의 2개의 오브젝트가 묶여져 combobox형태로 동작하도록 만들어 그 기능을 보여준다.
[Interaction]
: 입력창 또는 오른쪽 화살표버튼을 클릭하면 아래로 옵션항목이 리스트로 나오고 그 중 하나를 선택하면 선택값의 데이터를 리스트로 보여준다.

Code Description 

		

	
				
[autocomplete widget 사용]

				입력창에 텍스트를 쓸 때 지동완성 텍스트가 나오도록 하는 것과 같은 원리로, autocomplete widget을 사용하여 아무 텍스트를 쓰지 않았지만 focus되면 바로 모든 항목이 리스트로 나오게 할 수 있다.

				input창에 focus할 때, autocomplete의 search 이벤트가 발생하도록 강제로 만들어 적용시켜 모든 데이터 항목이 나오도록 한다.

				

			
			
[button의 아이콘, 모양, 위치]

				icon : button component의 옵션 중 하나로 icons 에 css에서 정의해둔 아이콘을 바꾸어 지정해 줄 수 있다. 

				button shape : 정사각형모양을 위해 기본적으로 적용된 라운드 처리를  css class들을 제거하거나 새로 다른 class를 추가해 줄 수 있다.(addClass, removeClass 사용)

				button position : 우선, css의 position 속성을 relative로 바꾸어 둔 후, position() 메소드를 사용하여, 입력창의 오른쪽에 버튼이 위치하도록 다음과 같이 지정해 줄 수 있다.(line #81-#84)
