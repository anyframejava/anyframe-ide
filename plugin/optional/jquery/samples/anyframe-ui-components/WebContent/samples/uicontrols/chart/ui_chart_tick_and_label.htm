Chart 눈금 및 레이블

: 차트의 눈금설정과 기본 레이블의 설정에 관해 조작하는 예이다.

Demo

Demo Description

배경 그리드

: 차트 뒷배경으로 존재하는 그리드에 대해, 가로, 세로 그리드 선을 show/hide 할 수 있다.

눈금 조절

: x, y축의 최소, 최대값, 눈금 간격을 지정할 수 있으며, 본 예에서는 y축에 대한 최소, 최대값과 눈금간격에 대해서 조작가능하도록 구현하였다.

  • 최소/최대값: 입력창에 숫자값을 입력하고 적용시키면, 그 값을 기준으로 하여 차트가 보여진다.
  • 눈금간격: 슬라이더가 움직임에 따라 그 간격이 조절된다.(슬라이더의 최소값은 1, 최대값은 20)

레이블 텍스트

: x축 전체에 대한 타이틀이름, y축 전체를 포함하는 타이틀 이름을 사용자임의로 입력하여 적용되도록 구현하였다.

눈금 레이블 각도

: 각 x, y축의 눈금별 레이블 또는 숫자에 대해 각도를 변경해 볼 수 있다.(타이틀의 명칭이 길어질 경우 쓰임)

기본 슬라이더 생성

				#58 - #60
				

				:  .slider() 코드로 기본 가로형태의 슬라이더를 생성할 수 있으며, orientation 옵션을 통해 방향을 설정할 수 있다.
				

				
				
단계(이동간격) 설정

				#62 - #65
				

				: step 옵션으로 이동간격 값을 설정할 수 있다. 단, 설정값은 숫자 상수 혹은 number 타입의 변수여야 한다.
				

				
				
범위설정형 슬라이더 : range 옵션

				

				: range 옵션으로 범위형 슬라이더를 생성할 수 있으며, 값의 형태는 다음과 같다.
				

				
    
					#72 - #88
					
  • "min"
					: 슬라이더의 가장 왼쪽에 있는 최소값을 시작점으로 고정하고, 움직이는 슬라이더 핸들러까지를 범위로 설정하는 형태로 생성된다.
					
    • 
					
  • "max"
					: 슬라이더의 가장 오른쪽에 있는 최대값을 시작점으로 고정하고, 움직이는 슬라이더 핸들러까지를 범위로 설정하는 형태로 생성된다.
					
    • 
					#100 - #103
					
  • true
					: range 옵션을 true로 초기설정을 하면 범위를 컨트롤할 수 있는 두개의 핸들러가 생성되고, values 옵션으로 값도 2개를 각각 설정할 수 있다.
					
    • 
				

				
				
value / values

				

				: 핸들러가 하나인 경우, value 옵션으로 값을 1개 설정하고, 핸들러가 두개 필요한 경우, values 옵션으로 2개를 배열로(다음과 같은 형태로) 설정해 주어야 한다.
				

				
$(ENV.ID_SLIDER_RANGE_TWO).slider({
     range : true,
     values : [10, 65]
});


				

				
				
range 동적설정

					

					:  범위형 슬라이더로 range : "min" 또는 "max" 값으로 동적으로 바꾸고자 재설정할 때, 값이 제대로 바뀌지 않는다. 
					필요한 경우, 직접 클래스에 접근하여 해당클래스를 제거하고 재설정하며, 값을 0 또는 100으로 이동시켜 다시 이동하도록 해야한다.
