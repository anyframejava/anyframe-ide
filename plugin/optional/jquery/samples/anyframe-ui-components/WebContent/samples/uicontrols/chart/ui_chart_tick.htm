Chart 눈금/그리드 조절

: 차트의 눈금과 그리드에 관한 설정에 관해 조작하는 예이다.

Demo

Demo Description

배경 그리드

: 차트 뒷배경으로 존재하는 그리드에 대해, 가로, 세로 그리드 선을 show/hide 할 수 있다.

눈금 조절

: x, y축의 최소, 최대값, 눈금 간격을 지정할 수 있으며, 본 예에서는 y축에 대한 최소, 최대값과 눈금간격에 대해서 조작가능하도록 구현하였다.

  • 최소/최대값: 입력창에 숫자값을 입력하고 적용시키면, 그 값을 기준으로 하여 차트가 보여진다.
  • 눈금간격: 슬라이더가 움직임에 따라 그 간격이 조절된다.(슬라이더의 최소값은 1, 최대값은 20)

Code Description 

				

				
배경 그리드

				

				:  초기설정으로 x축과 y축의 옵션에서 showGridline 을 true로 미리설정 후, 동적으로 바꾸어야 할 경우 setGrid 함수에서와 같이
				 target.options.axes.xaxis.tickOptions.showGridline 옵션을  value(x, y, xy 또는  both) 의 값으로 재설정한 후, replot함수로 다시 차트를 그려주도록 한다.
				

				
				
눈금 최소/최대값

				

				: target.options.axes.yaxis.min 또는 .max 옵션을 사용자 입력창으로 받은 값으로 재설정한 후 replot으로 다시 차트를 그려준다.
				

				
				
눈금 간격

				

				: target.options.axes.yaxis.tickInterval 옵션을 슬라이더 움직임에 따라 1- 20의 value 값을 받아 눈금간격이 재설정되도록 한다.
