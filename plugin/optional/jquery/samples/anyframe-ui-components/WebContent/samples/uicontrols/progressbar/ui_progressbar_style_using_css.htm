Component - Progressbar

기본 CSS를 이용한 Progressbar의 스타일 변화

: progressbar 의 기본 제공 CSS를 바꾸어 customizing 할 수 있는 방법을 제공한다.

Demo

Demo Description

스타일 변화

: border, value color, label position을 변화시켜 스타일을 조작할 수 있다.

  • border: thick, thin, none으로 border의 유무, 너비정도를 조작할 수 있다.
  • value color: solid 버튼을 통해 단색으로, gradation 버튼을 통해 색의 그라데이션으로 변화하는 것을 확인할 수 있다.
  • label position: progressbar에서 내부 라벨의 위치를 left, center, right 으로 바꾸어 조작해 볼 수 있다.

Code Description 

				

				
CSS Class 구조

				

				바의 외부 div, value 값의 내부 div, label의 div 각각의 위치와 class 구조는 다음과 같다.
				

				
				
    
					
  • 외부 division
					: id= "porgressbar" 로 설정되어 전체 크기 윤곽선(border)을 가지는 부분으로, 
					컴포넌트 생성시 .ui-progressbar의 클래스가 적용된다.
    • 
					
  • 내부 label division
					: id="progressbarLabel" 로 설정된 부분으로, label이 쓰여지는 division 공간이다. 
    • 
					
  • 내부 value division
					: progressbar의 컴퍼넌트를 생성하면 자동으로 내부에 .ui-progressbar-value클래스가 적용되며,
					값이 변화함에 따라 width가 value값의 %단위로 정해지게 된다.
    • 
				

				
				
CSS 변경

				

				border, value color, position 에 대해 어느 division에 접근하여 무엇을 바꿔야 하는지에 대한 방법은 다음과 같다.
				

				
				
    
					
  • border
					: id가 progressbar로 만든 division의 .ui-progressbar 클래스가 바의 border를 생성하게 되는데, 
					이를 css() 메소드를 통해 border값을 thick, thin, none 과 같은 값으로 변경할 수 있다.
					
    • 
					
  • value color
					: 외부 division 안에 자동적으로 생성되는  내부 value division의 css 중 background-color속성를 바꾸어 주면 된다.
					
    • 
					
  • label position
					: 외부 division 안에 label div을 쓰고, position을 absolute, width: progressbar의 크기로 설정한 후, 
					text-align 속성을 위치에 따라 left, center, right 으로 값을 바꾸어 주면 된다.
					
    •
