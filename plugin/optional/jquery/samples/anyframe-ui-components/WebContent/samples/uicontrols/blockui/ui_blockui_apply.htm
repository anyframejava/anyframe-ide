응용 기능


            

                : BlockUI 을 다양한 방법으로 활용할 수 있다.
            


            #55
            
  • $.growlUI()
                : Growl 형식의 BlockUI 를 출력한다. blockUI 라이브러리 기본제공 기능이다. 파라미터로 내용입력이 가능하다.
            
    • 
            
    
                
    해당 기능은 아래와 같이 풀어서 구현할 수 있다.
//CSS
div.growlUI { background: url(images/check48.png) no-repeat 10px 10px }
div.growlUI h1, div.growlUI h2 { color: white; padding: 5px 5px 5px 75px; text-align: left }
div.growlUI h2 { font-size: medium }

//Script
$.blockUI({
    message: $('div.growlUI'),
    fadeIn: 700,
    fadeOut: 700,
    timeout: 2000,
    showOverlay: false,
    centerY: false,
    css: {
        width: '350px',
        top: '10px',
        left: '',
        right: '10px',
        border: 'none',
        padding: '5px',
        backgroundColor: '#000',
        '-webkit-border-radius': '10px',
        '-moz-border-radius': '10px',
        opacity: .6,
        color: '#fff'
    }
});

//HTML
<div class="growlUI" style="display:none">
    <h1>Growl Notification</h1>
    <h2>Have a nice day!</h2>
</div>
