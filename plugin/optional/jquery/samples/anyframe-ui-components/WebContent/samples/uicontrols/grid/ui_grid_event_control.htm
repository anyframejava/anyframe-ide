Grid Event 처리

: Grid 내 다양한 이벤트 처리에 대한 예를 보여준다.

Demo

Demo Description

각 Event 발생 시 넘겨주는 파라미터 값을, 이벤트 로그에서 확인할 수 있다.

Event 설명

  • onSelectRow: row 클릭 시 발생된다.
  • ondblClickRow: row 를 더블클릭 시 발생된다.
  • onHeaderClick: Grid 상단에 있는 Caption Bar 우측에 있는 버튼을 클릭 시 발생된다.
  • onRightClickRow: row 를 마우스 우측버튼으로 클릭 시 발생된다.
  • beforeSelectRow: row 클릭 시 발생되며, onSelectRow 이전에 발생된다. 이벤트 처리 결과값에 따라서 선택여부를 결정한다.
