Demo Description

Dialog 팝업

: Dialog 를 팝업으로 활용하여, 특정 페이지를 띄우는 방법을 보여준다.
: 팝업이 출력되는 동안 Modal 옵션을 통해 Background 접근을 막을 수 있다.
: Dialog 'Open' & 'Close' 시 애니메이션 효과를 적용할 수 있다.

팝업 활용

		
  
		: Dialog 내부에 다른 페이지를 팝업처럼 출력할 수 있다.
		

		
    
            #26
			
  • autoOpen
			    : Dialog 자동으로 출력여부 설정
    • 
            #27
            
  • title
                : Dialog 타이틀 설정
    • 
            #28 - #29
            
  • width, height
                : Dialog 의 너비와 높이를 설정 (Auto 의 경우 내부 페이지 크기 기준으로 설정됨)
    • 
            #30
            
  • modal
                : True 의 경우, Dialog 외부는 Block (Modal Popup)
    • 
            #31
            
  • resizable
                : Dialog 크기 조절 가능여부 설정 
    • 
            #32
            
  • draggable
                : Dialog 드래그 가능여부 설정
    • 
            #33
            
  • closeOnEscape
                : 'ESC' 키를 눌렀을 때, Dialog Close 여부설정
    • 
            #34 - #35
            
  • show, hide
                : Dialog 가 Open 또는 Close 될때의 효과를 설정 (Dialog 애니메이션 효과 참고)
    • 
            #39
            
  • load([URL])
                : URL의 페이지를 dialog 안으로 출력한다.
