Component - Dialog

Dialog 최대/최소화

: dialog로 로그인창이 열리고 정해진 데이터(ID/PW) 를 입력, 출력하는 기능을 조작하는 예를 보여준다.

Demo

Demo Description

[Purpose]
: 다이얼로그를 통해 정해진 데이터를 입/출력하고, 간단한 데이터의 유효성 체크를 조작해 본다.
[Interaction]
: open버튼으로 다이얼로그 창을 열어 ID/PW 값을 입력하고, 정해둔 테스트 데이터 값과 일치하면 해당 정보를 나타내어 준다. (일치하지 않으면 빨간색 보더로 다시 입력하라는 정보가 나온다.)

Code Description 

		

	
			

				val() 메소드
 : val() 메소드는 매칭되는 요소의 현재값을 얻는다. 본 예제에서는 이를 이용하여 다이얼로그 상의
				input 값을 val() 메소드로 얻어서 그 값을 정해둔 데이터 값과 비교하여 간단한 유효성을 체크하며, 입력값과의
				일치여부에 따라 다른 메시지가 메인화면에 나오도록 한다.
