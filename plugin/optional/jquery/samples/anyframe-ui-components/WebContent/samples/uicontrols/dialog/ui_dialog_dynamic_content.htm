설문형식의 다이얼로그

: 하나의 dialog 안에서 사용자 응답에 따라 어떻게 내부 콘텐츠가 동적으로 생성되며 데이터를 조작할 수 있을지를 보여준다.
: 본 예제에서는 텍스트, 버튼, 라디오버튼을 이용한 간단한 설문응답형 dialog를 구현하였다.
: start버튼을 누르면 다이얼로그가 뜨고, 질문에 대해 yes 혹은 no 버튼으로 선택하면 그 결과 내용이 달라지게 된다.
: yes와 같은 경우, 라디오버튼으로 세부내용을 선택하도록 하며, no와 같은 경우, 설문이 종료된다.
