Edit Formatter

: Edit 입력창에서 화폐단위형식, 번호 단위형식을 자동으로 맞추어 주는 기능을 보여주는 예시이다.

Demo

Demo Description

숫자 형식화: Formatter

: 입력창에서 사용자의 입력값을 특정형식에 맞게 자동으로 포맷을 맞추어 준다.

  • 화폐단위 형식 : comma(",")가 세자리수마다 존재하여 화폐단위 형식으로 자동으로 포맷을 맞추어 준다.
  • 숫자단위 형식 : hyphen("-")이 세번째, 다음 네번째, 다음 네번째 숫자사이에 존재하여 휴대전화번호 형식으로 자동으로 맞추어 준다.

Code Description 

		

		
Autocomplete : search 이벤트

		#85 - #87 
		#93 - #95 
		

		:  입력값에 대해 autocomplete의 데이터를 검색하기 전에 발생되는 이벤트로, 
		본 예제에서는 자동으로 형식을 맞추어주는 동작을 이 search 이벤트를 이용하여 구현하였다.

		


		
Formatter 값의 반환

		#23 - #29
		#52 - #57 
		

		:  현재 입력창에 들어온 입력값을 val()함수로 얻어서 넘겨주면 이를 각 _moneyFormatter, _phoneFormatter 함수를 거쳐 출력값을 반환하여 입력창에 적용되도록한다.(val()함수 이용)
		

		
    
				
  • 
					_moneyFormatter
					: 한글자단위로 가장 뒤의 숫자부터 하나씩 얻어(charAt()이용) 숫자 3개 후에 네번째에 comma"," 가 들어가도록 구현하였다.
				
    • 
				
  • 
					_phoneFormatter
					: 한글자단위로 가장 첫번째 숫자부터 하나씩 얻어(charAt()이용) 세번째 숫자 후, 7번째 숫자 후에  hyphen"-" 가 들어가도록 구현하였다.
				
    • 
				
  • 
					
					: (또한, 정규식을 이용하여 숫자값만 들어오도록 체크하였음)
