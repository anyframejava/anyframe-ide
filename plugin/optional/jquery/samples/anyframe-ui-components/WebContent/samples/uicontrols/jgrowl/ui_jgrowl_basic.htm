Component - jGrowl

jGrowl 기본기능

: Animation 효과를 이용하여 알림창을 출력한다.

Demo

Demo Description

기본 기능

: Default jGrowl. 두번 이상 호출시 자동으로 아래로 추가된다. (Default)

  • Default : 기본 알림메시지를 오른쪽 상단(default)에 출력한다.

위치 변경

: Default jGrowl. 출력되는 위치를 변경할 수 있다.

  • (각 버튼) : 기본 알림메시지를 각 버튼이 지정하는 위치로 출력한다.

테마 변경

: jGrowl의 CSS를 변경하여 출력한다.

  • iPhone Style: CSS에 iPhone 이라는 이름으로 지정된 속성을 출력.
  • Smoke Style: CSS에 Smoke 이라는 이름으로 지정된 속성을 출력.
