Button 마우스 이벤트

: 버튼을 마우스로 클릭, 오버 하는 인터랙션을 할 때 일어날 수 있는 여러 변화를 조작해 볼 수 있다.

  • mouse click event 마우스로 이 버튼을 클릭하면, alert로 창이 띄워져 클릭동작이 이루어졌음을 확인할 수 있다.
  • mouse over event
    • text : 마우스 오버를 하면 버튼 내 텍스트가 바뀌며, 마우스 아웃하면 다시 처음상태의 텍스트로 되돌아온다.
    • color : 마우스 오버를 하면 버튼의 배경색이 바뀌어지며, 마우스 아웃하면 다시 처음상태로 되돌아 온다.
    • effect : 마우스 오버를 하면 버튼이 fade out 의 효과가 나타나며, 아웃하면 fade in 의 효과가 나타난다.

Code Description 

		

		
Click 이벤트

		
: 클릭하는 인터랙션으로 일어나는 동작에 대해서 다음과 같은 3가지 방법을 사용할 수 있다.

		

			
1. $(selector).click(eventHnadler)
2. $(selector).on('click', handler)
3. $(selector).trigger('click') 

		

		
		
Mouse Over 이벤트

		
 
		 : 마우스 오버라는 인터랙션으로 일어나는 동작에 대해서는 다음과 같은 3가지 방법을 사용할 수 있다.
		

		

			
1. $(selector).mouseover(handlerIn)/.mouseout(handlerOut)
2. $(selector).mouseenter(handlerIn)/.mouseleave(handlerOut) 
3. $(selector).hover(handlerIn, handlerOut)
* 1,2번과 같은 경우 $(selector).on() , $(selector).trigger() 와 같은 방식으로 사용가능

		

		
		
	mouseover/out과 mouseenter/leave 의 차이점

		
 
		 : .mouseover()/.mouseout() 와 .mouseenter()/mouseleave() 는 기본적으로 마우스오버라는 동작을 지원하는 이벤트함수이지만, 차이점이 존재한다.
		

		
    
			
  •  .mouseover()마우스포인터에 대해 child element까지 인지하여 마우스오버 인터랙션이 적용된다.
    
				div와 같은 한 블럭 안에 여러개의 또다른 내부 div  블럭이 있을 경우, 그 모든 포함된 블럭에 마우스가 위치하게 될 때마다 똑같은 마우스오버상태가 적용되어 이벤트가 발생하게 된다.
			
    • 
			
  •   .mouseenter()마우스포인터에 대해 한 개의 해당블럭만 인지하여 마우스오버 인터랙션이 적용된다. 
				하위의 어떤 child element가 있을지라도 반복적으로 적용되지 않는다.
			
    • 
		

		
		
 *위의 3번 hover는  mouseenter/mouseleave와 같은 방식으로 동작하게 된다.
