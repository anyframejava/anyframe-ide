Grid Edit Type 별 처리

: 컬럼 편집 시, Type 별로 입력 UI 를 다르게 구현하는 예를 보여준다.

Demo

Demo Description

Grid 컬럼의 Edit 시, 사용될 수 있는 UI 형태별로 데모를 구성하였다.
데모에서 볼 수 있는 것 처럼, 지원되는 각각의 형태는 html 의 "input" 태그의 type 과 같은 것임을 알 수 있다.

각 형태별로 Grid 를 구성하였고, 각각의 colModel 에 설정된 "edittype", "editoptions" 값에 따라 그 형태가 달라짐을 볼 수 있다.

Code Source


        




		

			
Code Description

			

			
Text Type

			#73
			

			
{name: "input", editable:true,  edittype:"text"} 			


			

			
			
TextArea Type

			#97
			

			
{name: "input", editable:true,  edittype:"textarea"}		


			
			

			
Select Type

			#122 ~ #127
			

			
{name: "input1",    editable:true,  edittype:"select", editoptions: { value: "opt1:선택1; opt2:선택2" } }, 
{name: "input2",    editable:true,  edittype:"select", editoptions: { value: {opt1:'선택1', opt2:'선택2', op3:'선택3' }  } }		


			


			
Checkbox Type

			#152 ~ #153
			

			
{name: "input",     editable:true,  edittype:"checkbox", editoptions: { value: "Check:UnCheck" } }		


			

			
			
Password Type

			#178 ~ #179
			

			
{name: "input",     editable:true,  edittype:"password", editoptions: { maxlength: 10 } } 


			
			
			
			
Button Type

			#204 ~ #205
			

			
{name: "input",     editable:true,  edittype:"button", editoptions: { value: "TestButton" } } 


			
			
			
			
Image Type

			#230 ~ #231
			

			
{name: "input",     editable:true,  edittype:"image", editoptions: { src: "sub/logo-jquery-ui.png" } } 


			
			
			
			
File Type

			#256
			

			
{name: "input",     editable:true,  edittype:"file"}


			

			
			

			위 각 타입별 코드에서 알 수 있듯이, "edittype" 과 "editoptions" 를 설정하여 원하는 편집 형태를 결정할 수 있다.
