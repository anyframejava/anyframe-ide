Demo Description

Dialog 기준위치변경

: 다이얼로그의 창이 뜨는 위치와 특정 공간을 기준으로 바꾸어 조작해 볼 수 있다.
: section 을 전체화면/ section #1/ section #2로 각각 바꾸어 보면, 해당 섹션이 기준공간이 되어 화면상에서 보여지는 위치가 달라지게 된다.
: 또한, location을 center/ top+left/ top+right/ bottom+left/ bottom+right 는기준섹션 안에서의 세부위치로 변경되는 부분이다.

Code Description 

		

		
		
위치 : Position 옵션

		
  
		: Position은 position : {my : "위치", at : "위치", of : "타겟"} 과 같은 방식으로 변경가능하다.
		

		
    
			
  • Section 
			: position 옵션의  "of" 로 타겟을 바꾸어 설정하면, 그 해당섹션 기준의 위치로 바뀌어 진다.
    • 
			
  • Location 
			: position 옵션의 "my", "at" 으로 다이얼로그 자신의 위치(my)가 타겟의 어느 위치(at)에 있도록 할지 설정하여 상대적인 위치를 조작할 수 있다.
    •
