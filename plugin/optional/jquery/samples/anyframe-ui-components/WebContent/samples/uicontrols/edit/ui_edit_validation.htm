Edit 유효성체크

: Edit 입력창에서 특정조건으로 유효성을 체크하는 기능을 보여주는 예이다.

Demo

Demo Description

유효한 값 체크

: 사용자가 입력창에 입력하는 값에 대해서 각 조건을 두고, 알맞지 않은 값은 처리하여 경고문을 보여주는 형식으로 이루어져 있다.
: 본 샘플에서 제공하는 단위조건은 값의 유무/ 영문 문자열/ 문자열의 길이/ 숫자값/ 숫자값의 범위/ 입력값의 일치여부/ 특정 값의 조합/ 이메일포맷 이다.

 체크방식

		

		: 사용자가 입력한 값과 각 조건에 알맞는 표현식으로 RegExp을 생성한 후, 그 값을 test() 메소드로 비교한다.

		그 결과,  true/false 의 결과값으로 나누어 경고문을 보여주는 방식으로 구현되어 있다.
