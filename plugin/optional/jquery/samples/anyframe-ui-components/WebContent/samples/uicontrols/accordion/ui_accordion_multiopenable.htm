하나 이상의 섹션이 활성화되어 필요한 부분을 멀티로 열어놓고 볼 수 있는 예이다.

Demo

Demo Description : 

 .....
[Pupose]
여러개의 헤더에 해당하는 내용들을 멀티로 열어두고 보아야 할 필요가 있을 경우 이와 같은 예시를 사용한다.
[Interaction]
각 헤더를 클릭하면 해당 섹션의 내용이 아래로 열리게 되며, 그 다음 다른 섹션의 헤더를 클릭해도 영향을 받지않고 열려진 채로 있게 된다.
(활성중인 한 섹션이 닫히게 하려면 다시 클릭하면 된다.)
