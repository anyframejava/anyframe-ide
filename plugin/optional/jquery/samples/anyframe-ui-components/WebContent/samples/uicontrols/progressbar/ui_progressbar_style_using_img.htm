Component - Progressbar

다양한 형태의 이미지를 이용한 Progressbar

: progressbar 의 형태를 customizing 할 수 있는 방법을 제공한다.

Demo

Demo Description

형태의 다양화

: 일반적인 네모박스형태의 progressbar와 달리 다양한 형태로 진행상태를 알려주는 bar를 확인해 볼 수 있다.

  • square shape type : 단순 직사각형의 진행바에 25% 진행시마다 그 위치에 정사각형 도형을 두어 그 정도가 강조하여 보여준다.
  • circle shape type : 단순 직사각형의 진행바에 25% 진행시마다 그 위치에 원을 두어 그 정도를 강조하여 보여준다.

진행값의 현재상태 다양화

: progressbar를 둘러싸는 border line 형태를 없애고, value 값이 누적되어 채워지는 형태가 아닌, 현재상태의 위치를 강조하여 보여줄 수 있다.

  • circle point type : 원의 도형이 현재상태를 가리키며 진행정도를 보여준다.
  • car point type : 차형태의 아이콘이 현재상태를 가리키며 진행정도를 보여준다.

Bar 이미지 바꾸기

				

				바의 외부과 내부 value값의 div에 각 기존 border를 없애고, 배경이미지로 customizing할 이미지를 넣는다.   
				

				
				
    
					
  • 외부 스타일
						
      
							
    • border : none
      • 
							
    • height : 이미지의 높이값
      • 
							
    • background : 바의 윤곽선이 나타나는 이미지 소스 경로 및 no-repeat 지정
      • 
						
    
					
    • 
					
  • 내부 스타일
						
      
							
    • border : none
      • 
							
    • margin : 0 (외부 윤곽선 이미지와 내부 이미지가 sync가 맞도록 설정하기 위함)
      • 
							
    • background : 바의 내부 색이 채워진 이미지 소스 경로 및 no-repeat 지정
      • 
						
    
					
    • 
				

				
				
Bar의 현재포인트 이미지 설정

				

				바의 외부 border 일부와 내부값의 div에 존재하는 모든 배경을 없애고, 내부적으로 새로운 div을 생성, 추가하여 이미지를 삽입한다.
				

				
				
    
					
  • 외부 스타일
						
      
							
    • border-width : (아래밑줄만 존재하도록 보더 너비 값을 준다.)
      • 
						
    
					
    • 
					
  • 내부 스타일
						
      
							
    • border : none
      • 
							
    • background : none
      • 
							
    • text-align : right(바가 움직일 때 항상 가장 오른쪽에 있도록 하기위함)
      • 
						
    
					
    • 
					
  • 현재포이트 이미지 추가
						
      
							
    • append() 메소드를 통해 div, img 태그를 삽입하여 이미지를 설정해 준다.
