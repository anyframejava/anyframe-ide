Chart Tooltip기능

: 차트에서 x축, y축 또는 특정 텍스트 정보를 각 꼭지점에 tooltip으로 나타나도록 하는 기능의 예이다.

Demo

Demo Description

Tooltip 정보

: 차트의 각 꼭지점에 마우스 오버시 나타나게 될 tooltip 정보를 선택한 값이 적용되어 나오게 된다.

tooltip 위치

: 값의 꼭지점을 중심으로 하여 tooltip이 나오게 될 위치를 설정할 수 있다.

tooltip 스타일

: 차트의 각 꼭지점에 마우스 오버시 나오는 tooltip에 대해 배경색과 텍스트색을 변경해 볼 수 있다.
