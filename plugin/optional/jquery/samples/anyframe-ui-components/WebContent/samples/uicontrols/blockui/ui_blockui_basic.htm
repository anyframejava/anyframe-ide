Component - BlockUI

BlockUI 기본기능

: 특정 Transaction 발생시 불필요한 화면의 조작을 막기위해 화면을 Block 한다.

Demo

Demo Description

기본 Block

: 서버의 Transaction 발생시 화면을 Block 한다. (Default)

  • Loading Message : 'Please wait..' 메시지를 출력하고 화면을 Block 한다.

이미지 Block

: 서버의 Transaction 발생시 화면을 Block 한다.(특정 이미지)

  • Image Message : 설정된 이미지를 출력하고 화면을 Block 한다.
